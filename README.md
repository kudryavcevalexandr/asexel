# Excel parser + browser view

Скрипт читает `.xls` файл и показывает в браузере:
- имя файла, путь, размер, дату изменения;
- количество листов;
- по каждому листу: количество строк/колонок и первые N строк.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск

По умолчанию скрипт ищет файл по пути из скриншота:
`~/Download/график ТЭЦ26 260424.xls`

```bash
python app.py
```

Если файл в другом месте:

```bash
EXCEL_PATH="/путь/к/график ТЭЦ26 260424.xls" python app.py
```

Открой в браузере: `http://127.0.0.1:5000`

## Полезные переменные

- `EXCEL_PATH` — путь до Excel-файла;
- `PREVIEW_ROWS` — сколько строк показывать в таблице (по умолчанию `10`).
